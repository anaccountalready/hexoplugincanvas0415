## 快速开始

### 什么是 hexo-canvas-doodle？

hexo-canvas-doodle 是一个为 Hexo 博客设计的插件，它允许你在文章中嵌入一个可交互的画布。读者可以在画布上用黑色画笔涂鸦，并且涂鸦内容可以保存到浏览器本地存储中。

### 核心功能

- ✏️ **黑色画笔涂鸦**：简单的黑色画笔，线宽适中
- 💾 **本地保存**：涂鸦内容自动保存到浏览器
- 📱 **多设备支持**：支持鼠标和触摸操作
- 🔄 **随时加载**：可以随时恢复之前保存的涂鸦

## 安装指南

### 前置要求

在安装之前，请确保你已经：

1. 安装了 Node.js (推荐 v12 或更高版本)
2. 有一个正在使用的 Hexo 博客项目
3. 了解基本的 Hexo 操作

### 安装步骤

#### 方法一：通过 npm 安装（推荐）

1. 打开终端，进入你的 Hexo 博客根目录：

```bash
cd /path/to/your/hexo/blog
```

2. 安装插件：

```bash
npm install hexo-canvas-doodle --save
```

3. 验证安装：

检查 `node_modules` 目录下是否有 `hexo-canvas-doodle` 文件夹。

#### 方法二：手动安装

如果你想使用开发版本或自定义修改，可以手动安装：

1. 下载插件文件（或从 GitHub 克隆）

2. 在你的 Hexo 博客的 `node_modules` 目录下创建 `hexo-canvas-doodle` 文件夹

3. 将所有插件文件复制到该文件夹中

4. 在博客根目录的 `_config.yml` 文件中添加：

```yaml
plugins:
  - hexo-canvas-doodle
```

### 验证安装

安装完成后，你可以通过以下步骤验证：

1. 在任意一篇文章中添加画布标签（详见使用方法）
2. 运行 `hexo clean && hexo server`
3. 访问文章页面，查看是否出现画布

## 使用方法

### 基本用法

在你的 Markdown 文章中，使用以下语法嵌入画布：

```markdown
{% canvas %}
```

这将创建一个默认大小（800x400像素）的画布。

### 自定义画布

你可以通过参数自定义画布的ID和尺寸：

```markdown
{% canvas [画布ID] [宽度] [高度] %}
```

#### 参数说明

| 参数 | 类型 | 必填 | 默认值 | 说明 |
|------|------|------|--------|------|
| 画布ID | 字符串 | 否 | 自动生成 | 用于标识画布，建议使用有意义的名称 |
| 宽度 | 数字 | 否 | 800 | 画布的宽度（像素） |
| 高度 | 数字 | 否 | 400 | 画布的高度（像素） |

### 使用示例

#### 示例1：默认画布

```markdown
# 我的文章

这是一个默认大小的画布：

{% canvas %}

你可以在上面随意涂鸦！
```

#### 示例2：自定义尺寸

```markdown
# 绘画练习

这里有一个更大的画布：

{% canvas practice-canvas 1000 600 %}

尽情发挥你的创意吧！
```

#### 示例3：多个画布

你可以在同一篇文章中嵌入多个画布：

```markdown
# 多画布示例

## 第一个画布

{% canvas canvas-1 600 300 %}

## 第二个画布

{% canvas canvas-2 600 300 %}
```

**注意**：每个画布应该使用不同的ID，否则它们会共享相同的存储数据。

## 功能详解

### 画布界面

每个画布组件包含以下部分：

1. **画布区域**：白色背景的绘图区域
2. **控制按钮**：位于画布下方的三个按钮
3. **状态提示**：显示操作结果的文本区域

### 绘图功能

#### 如何绘图

1. **鼠标操作**：
   - 点击并按住鼠标左键开始绘图
   - 移动鼠标进行绘制
   - 释放鼠标左键停止绘图

2. **触摸操作**（适用于手机和平板）：
   - 用手指触摸画布开始绘图
   - 移动手指进行绘制
   - 抬起手指停止绘图

#### 画笔特性

- **颜色**：纯黑色 (#000000)
- **线宽**：3像素
- **线条端点**：圆形
- **线条连接**：圆形

### 保存功能

#### 如何保存

1. 在画布上完成涂鸦后
2. 点击 **"保存"** 按钮
3. 状态提示会显示 "保存成功！"

#### 保存机制

- 数据保存在浏览器的 **localStorage** 中
- 每个画布使用独立的存储键（格式：`hexo-canvas-{画布ID}`）
- 数据以 **Base64 编码的 PNG 图片** 格式存储
- 只要不清除浏览器数据，保存的内容会一直存在

### 加载功能

#### 如何加载

1. 点击 **"加载"** 按钮
2. 系统会从本地存储中读取之前保存的内容
3. 画布会自动恢复到保存时的状态

#### 注意事项

- 如果没有保存过内容，会显示 "暂无保存的内容"
- 加载会**覆盖**当前画布上的所有内容
- 建议在加载前先保存当前内容（如果需要）

### 清除功能

#### 如何清除

1. 点击 **"清除"** 按钮
2. 画布上的所有内容会被立即清空
3. 状态提示会显示 "画布已清除"

#### 注意事项

- 清除操作**不可撤销**
- 清除不会影响已保存的内容
- 如果需要保存当前内容，请在清除前点击"保存"

## 常见问题

### Q1: 为什么我保存的内容不见了？

**可能的原因：**

1. **清除了浏览器数据**：如果清除了浏览器的缓存或Cookie，localStorage中的数据会被删除
2. **使用了不同的浏览器**：数据保存在特定浏览器中，换浏览器后无法访问
3. **使用了隐私模式**：隐私模式下的localStorage在关闭浏览器后会被清除
4. **画布ID不同**：确保使用相同的画布ID才能加载之前的内容

**解决方案：**

- 定期导出重要的涂鸦内容（可以右键画布保存为图片）
- 使用相同的浏览器和设备访问
- 避免在隐私模式下使用保存功能

### Q2: 画布在手机上显示太小怎么办？

**解决方案：**

1. **设置合适的尺寸**：在标签中指定较小的宽度，例如：

```markdown
{% canvas mobile-canvas 350 200 %}
```

2. **自定义CSS**：在主题的CSS文件中添加响应式样式：

```css
@media (max-width: 768px) {
  .hexo-canvas-doodle {
    max-width: 100% !important;
    height: auto !important;
  }
}
```

### Q3: 如何修改画笔的颜色和粗细？

**当前版本限制：**

hexo-canvas-doodle v1.0.0 只提供固定的黑色画笔（线宽3px），这是为了保持插件的简洁性。

**未来版本计划：**

我们计划在后续版本中添加自定义画笔功能，包括：
- 颜色选择器
- 线宽调节
- 橡皮擦功能

**临时解决方案：**

如果你需要自定义画笔，可以：
1. 直接修改 `assets/canvas-doodle.js` 文件中的画笔设置
2. 或者等待后续版本更新

### Q4: 保存的数据会占用多少空间？

**存储空间：**

- 每个画布保存为PNG图片的Base64编码
- 800x400像素的空白画布约占用 **2KB**
- 充满内容的画布约占用 **50-200KB**（取决于内容复杂度）
- 浏览器的localStorage通常有 **5-10MB** 的限制

**建议：**

- 不需要的保存内容可以通过清除画布后重新保存来覆盖
- 重要内容建议导出为图片文件

### Q5: 为什么在某些主题中画布显示不正常？

**可能的原因：**

1. **主题CSS冲突**：某些主题的全局样式可能影响画布
2. **布局问题**：主题的文章容器宽度可能限制画布显示

**解决方案：**

1. **检查CSS冲突**：使用浏览器开发者工具检查是否有样式覆盖
2. **自定义样式**：在主题的自定义CSS中添加：

```css
.hexo-canvas-doodle-container {
  box-sizing: border-box;
}

.hexo-canvas-doodle {
  box-sizing: border-box;
  max-width: 100%;
}
```

### Q6: 如何备份我的涂鸦内容？

**方法一：右键保存图片**

1. 在画布上右键点击
2. 选择 "保存图片" 或 "另存为"
3. 选择保存位置

**方法二：使用开发者工具**

1. 按 F12 打开开发者工具
2. 切换到 Console（控制台）标签
3. 输入以下代码（替换 `your-canvas-id` 为实际的画布ID）：

```javascript
const canvas = document.getElementById('your-canvas-id');
const link = document.createElement('a');
link.download = 'my-doodle.png';
link.href = canvas.toDataURL();
link.click();
```

## 高级配置

### 自定义样式

你可以通过在主题的CSS文件中添加自定义样式来修改画布的外观。

#### 修改容器样式

```css
.hexo-canvas-doodle-container {
  background-color: #f0f0f0;  /* 更改背景色 */
  border: 2px dashed #999;     /* 更改边框样式 */
  padding: 20px;                /* 更改内边距 */
  margin: 30px 0;               /* 更改外边距 */
}
```

#### 修改画布样式

```css
.hexo-canvas-doodle {
  border: 3px solid #333;       /* 更改边框 */
  border-radius: 10px;          /* 圆角 */
  box-shadow: 0 4px 8px rgba(0,0,0,0.1);  /* 添加阴影 */
}
```

#### 修改按钮样式

```css
/* 所有按钮 */
.hexo-canvas-btn {
  padding: 10px 20px;           /* 更改按钮大小 */
  font-size: 16px;               /* 更改字体大小 */
  border-radius: 20px;           /* 圆角按钮 */
}

/* 清除按钮 */
.hexo-canvas-clear {
  background-color: #ff6b6b;
}

/* 保存按钮 */
.hexo-canvas-save {
  background-color: #51cf66;
}

/* 加载按钮 */
.hexo-canvas-load {
  background-color: #ffd43b;
  color: #333;
}
```

### JavaScript API

插件提供了全局的 JavaScript API，可以通过代码控制画布。

#### API 方法

```javascript
// 保存指定画布
HexoCanvasDoodle.save('canvas-id');

// 加载指定画布
HexoCanvasDoodle.load('canvas-id');

// 清除指定画布
HexoCanvasDoodle.clear('canvas-id');

// 初始化新画布（动态添加的画布）
HexoCanvasDoodle.init('new-canvas-id');
```

#### 使用示例

```html
<!-- 自定义按钮 -->
<button onclick="saveMyCanvas()">自定义保存</button>

<script>
function saveMyCanvas() {
  if (HexoCanvasDoodle.save('my-canvas')) {
    alert('保存成功！');
  } else {
    alert('保存失败！');
  }
}
</script>
```
