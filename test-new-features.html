📋 新增功能列表

💡 提示： 本次更新新增了无限画布和迷你地图导航两大功能！

✨ 功能一：无限画布（动态扩展）

  • 不再受固定尺寸限制：画布可以无限扩展，无论画到哪里都有空间
  • 智能边界检测：当绘制接近当前画布边界时，自动扩展画布尺寸
  • 双倍扩展策略：每次扩展时将画布尺寸翻倍，确保流畅体验
  • 支持负坐标：可以向左和向上绘制到负坐标区域

🗺️ 功能二：迷你地图导航

  • 实时缩略图：画布右下角显示迷你地图，实时展示所有绘制内容
  • 视口指示器：红色矩形表示当前可见区域在整个画布中的位置
  • 点击定位：点击迷你地图任意位置，画布视图自动跳转到对应位置
  • 拖拽导航：在迷你地图上拖拽可以快速浏览整个画布
  • 智能边界：迷你地图只显示有内容的区域，自动忽略空白区域

⚠️ 测试说明： 请在下方画布上测试以下场景：

1. 在画布边缘绘制，观察画布是否自动扩展

2. 绘制多个分散的图形，观察迷你地图的显示

3. 点击迷你地图的不同位置，验证定位功能

4. 缩放和平移画布，观察迷你地图中的视口指示器变化

📝 操作指南

🎨 基础操作

  • 绘制：鼠标左键拖拽
  • 平移画布：按住鼠标中键拖拽，或按住空格键+左键拖拽
  • 缩放画布：鼠标滚轮，或点击 +/- 按钮
  • 添加文本：输入文本后点击"添加文本"按钮，然后点击画布放置

🗺️ 迷你地图操作

  • 查看缩略图：画布右下角的迷你地图显示所有绘制内容
  • 红色矩形：表示当前可见区域
  • 点击定位：点击迷你地图任意位置跳转到对应区域
  • 拖拽导航：在迷你地图上拖拽快速浏览

♾️ 无限画布特性

  • 向右/向下绘制：画布自动扩展，无需担心边界
  • 向左/向上绘制：支持负坐标区域，同样自动扩展
  • 保存/加载：保存时只裁剪有内容的区域，加载时自动居中
